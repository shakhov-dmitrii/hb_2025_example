# Проект пример

Данный проект сделан, чтобы показать, как простыми решениями добиться максимальной пользы от автотестов.

## Технологии
1. Python
2. Pytest
3. Requests
4. Allure
5. ruff
6. Github Actions

## Инструкции по установке и запуску

Необходимо установить UV https://docs.astral.sh/uv/getting-started/installation/

### Установка зависимостей

Убедитесь, что у вас установлен `uv` и все необходимые зависимости:

```bash
uv sync --dev
```

### Запуск тестов
Для запуска тестов используйте команду:

```bash
make test
```

## Как пользоваться

Часть информации дана в докладе =)
А так - исследуйте проект и приходите с вопросами.

### Нотификации
Все уведомления по умолчанию падают в канал https://t.me/hb_example_notification
Для получения своего токена для отправки уведомлений необходимо получить токен. Сделать это можно с помощью [BotFather](https://core.telegram.or
Сгенерированный токен используем в переменной окружения ```TG_BOT_TOKEN```

### Получение токена CoinGecko
1. Регистрируемся и авторизуемся на https://www.coingecko.com/
2. Открываем https://www.coingecko.com/en/developers/dashboard
3. Создаем новый токен кликнув "Add New Key"
